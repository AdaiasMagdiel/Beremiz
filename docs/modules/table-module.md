# Table Module

The `table` module provides methods for manipulating tables in Beremiz.

## Available Methods:

- [**`length`**](#length)

---

## `length`

Returns the number of elements in the table.

**Usage**:

```beremiz
[1, 2, 3, "a", "b", "c"] table.length
```

**Example**:

```beremiz
[1, 2, 3, "a", "b", "c"] table.length show    # Output: 6
```
