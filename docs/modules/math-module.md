# Math Module

The `math` module provides methods for math manipulation.

## Available Methods:

- [**`toint`**](#toint)

---

## `toint`

Converts a valid string value into a number.

**Usage**:

```beremiz
"value" math.toint
```

**Example**:

```beremiz
"10" math.toint
5 + show          # Output: 15
```
